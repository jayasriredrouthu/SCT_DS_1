# 📊 SkillCraft Technology – Data Science Internship

## Task 01: Data Visualization using Python

### Project Overview

This project was completed as part of the **SkillCraft Technology Data Science Internship – Task 01**.

The objective of this project is to create a **Bar Chart** and **Histogram** to visualize the distribution of categorical and continuous variables using Python. The analysis was performed on the **Indian Developer Burnout Dataset**.

---

## Task Objective

* Create a Histogram to visualize the distribution of Age.
* Create a Bar Chart to visualize the distribution of Gender.
* Perform basic data cleaning and preprocessing.
* Understand the distribution of categorical and continuous variables through data visualization.

---

## Dataset

* **Dataset:** Indian Developer Burnout Dataset (2026)
